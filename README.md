#  Web E-Commerce Product Page

A responsive front-end product page built from scratch using HTML, CSS, and JavaScript — no frameworks or libraries.

##  Built With
- **HTML5** — semantic structure
- **CSS3** — Flexbox & Grid layout, transitions, hover states
- **JavaScript** — add-to-cart interaction, dynamic pricing display

##  Features
- Responsive layout — works on desktop and mobile
- Product image gallery
- Dynamic pricing and quantity selector
- Add-to-cart button with interactive feedback
- Accessible colour contrast and keyboard-friendly design

##  What I Learned
- Building responsive layouts with CSS Flexbox and Grid
- Writing clean, semantic HTML5
- Handling DOM events and user interactions with vanilla JavaScript
- Applying UX principles like visual feedback and accessibility
